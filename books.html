Main resources: lecture notes and textbook

Number Theory lecture notes

The main resource for the Number Theory part of the unit is a set of lecture notes available as a pdf.

Number Theory, by Killian O'Brien.

Worked solutions to the exercises from the Number Theory notes.

These lecture notes contain the definitions, discussions, proofs and examples for the topics we will study. Each chapter contains exercises and problems that will form the basis of our work in tutorials.

Abstract Algebra textbook

The main resource for the Abstract Algebra part of the unit is the open-source textbook 

Abstract Algebra: Theory and Applications, by Tom Judson and Robert Beezer.

This is a large book with 23 Chapters but in our unit we will only be studying a selection of the initial chapters on group theory. These are chapters 3,4,5,6,9 and 10. (see the schedule below for more detail).

Below you will find links to the online dynamic html version of the book and a static pdf covering our chapters. For general use I recommend using the first link to the online version, which works well on the smaller screens smartphones and tablets as well as laptops and PCs. But the alternative pdf version is provided in case you prefer working with that.
